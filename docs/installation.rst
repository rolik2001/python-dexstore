Installation
************

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python3-dev
    $ pip3 install dexstore

Manual installation:

::

    $ git clone https://github.com/xeroc/python-dexstore/
    $ cd python-dexstore
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip3 install dexstore --user --upgrade
